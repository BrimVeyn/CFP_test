# Installation
``` bash
git clone https://github.com/hckforpeace/CFP_test
```

## Run backend
* Open a new Terminal
``` bash
cd backend && npm run start
```

## Run frontend 
* Open a new Terminal
``` bash
cd frontend && npm run dev
```

## Tech Stack
* backend (node.js / fastify / Typescript)
* frontend (node.js / react.js / Typescript)
